### AXIOM sMM
This repository contains a Julia-based implementation of the SMM-module from AXIOM paper using RxInfer.jl

Main file is the ipynb notebook.

This work is based on:
1.  **AXIOM Paper:** [Active Inference and Object-Centric Modeling](https://arxiv.org/abs/2505.24784)
2.  **Base Implementation:** Inspired by the [Online Message Passing Dirichlet Process](https://github.com/biaslab/OnlineMessagePassingDirichletProcess/tree/main) implementation by Biaslab.
